# Impeccable-UI-Kit

Local-first multi-framework UI kit MVP.

## Commands
- `npm test` — run Node built-in tests
- `npm start` — launch the local API + browser preview on port 4492

## Conventions
- Use CommonJS + Node built-ins only for the MVP slice
- Keep the browser catalog zero-build (single server-rendered HTML page)
- Add or update tests before changing component-pack generation logic
